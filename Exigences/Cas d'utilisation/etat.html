Cas d'utilisation : Afficher l'état des mes robots.

But : Faire référence à l'évolution des données émises par les robots.

Préconditions :

  • L'utilisateur a déjà un compte.
  • L'utilisateur a déjà une liste de robots a gerer.

Acteurs : Utilisateur.

Scénario principal :

  1. L'utilisateur insère les données de son compte.
  2. Le système vérifie l'information.
  3. Le système obtienne la liste de robots liée à cet utilisateur.
  4. L'utilisateur demande l'information liée à chaque robot (position, vitesse, niveau de batterie, consommation CPU et mémoire).
  5. Le système affiche l'information demandée.
  6. L'utilisateur finit la session.
  7. Le système confirme la fermeture de session.

Scénario alternatif :

  • 5.1 L'utilisateur demande l'historique des données de chaque robot.
  • 5.2 Le système affiche l'information demandée.
  • 5.3 Le scénario reprend à l'étape 6.

Scénario alternatif :

  • 1.1 L'utilisateur insère les données incorrectes de son compte.
  • 1.2 Le système affiche un message d'erreur.
  • 1.3 Le scénario se termine
